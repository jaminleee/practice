**과제 제출 연습용입니다.**

### 과제 순서
1. 각 주차 세션을 맡은 운영진이 Issue에 과제를 올린다
2. 아기사자는 본인의 local에서 과제를 진행한 뒤, 기한 내에 Pull Request를 보낸다.
3. 각 아기사자의 멘토는 Wiki에 아기사자의 과제 현황을 체크한다
